About the Program

  • About the Program

With our rigorous and personalized curriculum, we will prepare you for a career in tech.

Our coding school offers premium instruction aimed to help you succeed in the tech industry. We teach the leading front end technologies and programming skills required for quality web development. We also organize collaborative projects that will promote successful team interactions in real world settings.

With a small teacher to student ratio size, we cater to you on a personal level. We have strategically developed a packed curriculum to prepare you for the workplace. While we follow our schedule, we also are flexible enough to work with you on an individual level to ensure that you get the extra help and practice that you need to absorb the material.

Curriculum

Programming Basics, HTML, CSS, Bootstrap

Week 1 through 2
Learn to build elegant websites with HTML and CSS and apply advanced styling methods. Then create mobile-friendly web pages very quickly with Twitter Bootstrap framework.

Javascript Foundation

Week 3
Learn the building blocks of Javascript and find out its interactive capabilities as the leading programming language on the web. Use Javascript to create applications that the users can interact with.

Advanced Javascript and Beginning to Advanced Angular

Week 4 and 5
Learn advanced Javascript methods, and build web applications to meet a real-world solution. Applications may have authorized login, different permissions, ability to create, edit, and delete records.

Job Search and Career

Week 6
Learn what it is like to be a web developer or software engineer from current working professionals.Showcase your ability with a stunning portfolio website and samples of your projects. Craft impactful resumes and practice mock interviews and negotiation skills that are applicable in all industries. Become a top job candidate, and get in touch with technical recruiters.
